# Parking Tickets Interactive Dashboard

This project uses a  parking ticket dataset, cleaned it to build an interactive Streamlit dashboard.

## Features
- Big-picture summary metrics
- Tickets by district
- Tickets by month
- Top streets by ticket count
- Top offense types
- Interactive filters for district, street, state, and offense description
- Ticket location map
- Detailed ticket table

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
